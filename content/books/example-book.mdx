# 예시 책 제목

이것은 `content/books` 폴더에 있는 예시 책 파일입니다.

⸻

## 서재 기능 소개

서재 페이지는 `content/books` 폴더의 MDX 파일들을 표시합니다.

### 주요 기능

1. **책 목록 표시**: 모든 책을 그리드 레이아웃으로 표시합니다.
2. **책 상세 페이지**: 각 책을 클릭하면 `/books/[slug]` 페이지로 이동합니다.
3. **이미지 지원**: 책 표지 이미지를 표시할 수 있습니다.

### 이미지 사용 방법

프론트매터에서 `image` 필드를 사용하여 책 표지 이미지를 지정할 수 있습니다:

```yaml
---
title: 예시 책 제목
image: /images/me.jpeg
---
```

또는 외부 URL도 사용 가능합니다:

```yaml
---
title: 예시 책 제목
image: https://example.com/book-cover.jpg
---
```
